# FlowForge

A modular workflow automation platform built entirely in Python.

## Features

- Workflow Engine
- Plugin System
- Scheduler
- Background Workers
- Event Triggers
- Conditional Flows
- File Processing
- Database Tasks
- HTTP Requests
- Logging
- Retry Policies
- REST API
- CLI

## Architecture

FlowForge is built around a node-based execution engine where workflows are represented as directed graphs.

Each node is responsible for executing a single action while sharing a common execution context across the workflow.
